
# Calendar Application

## Demo
[Live Demo](https://calender-eight-xi.vercel.app/)

## About
This is a calendar application assignment from **takeUforward**.

## Features
- View today's date
- Add specific notes to any date
- Aesthetic UI with animations using Framer Motion
- Responsive and user-friendly interface

## Project Structure
```
├── calendar.jsx (Main component)
├── CalendarHeader.jsx
├── CalendarGrid.jsx
└── NotePopup.jsx
```

## Installation & Setup

1. **Clone the repository**
    ```bash
    git clone <repository-url>
    cd calendar
    ```

2. **Install dependencies**
    ```bash
    npm i
    ```

3. **Start development server**
    ```bash
    npm run dev
    ```

4. Follow the localhost link provided in the terminal

## Usage
- Navigate through months using the header controls
- Click on any date to open the note popup
- Add or edit notes for specific dates
- Today's date is highlighted

## Code Readability
- Modular file structure for better maintainability
- Separated concerns: Header, Grid, and Popup components
- Clean and consistent code formatting
